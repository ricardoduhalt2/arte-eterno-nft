Instrucciones para probar

Este archivo HTML es independiente y no requiere ejecutar la aplicación React. Para probarlo:

  1. Abre este archivo directamente en tu navegador (doble clic en el archivo)
  2. Si ves el carrusel animado funcionando correctamente aquí pero no en la aplicación React, el problema podría estar relacionado con:
    • Conflictos de CSS en la aplicación principal
    • Problemas con la carga de imágenes desde IPFS
    • Problemas de rendimiento en la aplicación React
  3. Para limpiar la caché del navegador:
    • Chrome: Ctrl+Shift+Del (Windows/Linux) o Cmd+Shift+Del (Mac)
    • Firefox: Ctrl+Shift+Del (Windows/Linux) o Cmd+Shift+Del (Mac)
    • Safari: Cmd+Option+E
  4. Después de limpiar la caché, reinicia el navegador y vuelve a cargar la aplicación
